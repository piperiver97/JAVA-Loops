<h1>Java - Loops</h1>
<p>Crea un programa que dado un número entero, n, imprima su tabla de multiplicar (del 1 al 10). (No es necesaria la salida por terminal)

<li>5 x 3 = 15</li>
<li>5 x 4 = 20</li>
<li>5 x 5 = 25</li>
<li>5 x 6 = 30</li>
<li>5 x 7 = 35</li>
<li>5 x 8 = 40</li>
<li>5 x 9 = 45</li>
<li>5 x 10 = 50</li>
